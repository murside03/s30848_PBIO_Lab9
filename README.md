# FASTA DNA Sequence Generator
**Album number:** s30848

Random DNA sequence generator in FASTA format.

## Features
- Random DNA sequence generation
- Input validation (length, ID)
- Name embedding in sequence
- Nucleotide statistics & GC-content
- Motif search
- Complement & reverse complement
- DNA -> mRNA transcription
- Sliding window GC analysis (CSV output)

## Usage
python S30848.py

## Requirements
Python 3.10+
